## 🚀 Getting Started

### ☑️ Prerequisites

Before getting started with TEMP-MAIL.git, ensure your runtime environment meets the following requirements:

- **Programming Language:** TypeScript
- **Package Manager:** Npm


### ⚙️ Local Installation

Install TEMP-MAIL.git using one of the following methods:

**Build from source:**

1. Clone the TEMP-MAIL.git repository:
```sh
❯ git clone https://github.com/Nayan-withhat/TEMP-MAIL.git
```

2. Navigate to the project directory:
```sh
❯ cd TEMP-MAIL.git
```

3. Install the project dependencies:


**Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm install
```




### 🤖 Usage
Run TEMP-MAIL.git using the following command:
**Using `npm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/npm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://www.npmjs.com/)

```sh
❯ npm run dev
```
